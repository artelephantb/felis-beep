Felis Beep - Scratch Script

Felis Beep is an extension for Firefox, to add BeepBox into Scratch Script!

Usage

For usage in BeepBox, use the BeepBox website (you may have to scroll down).

Save Button

This is the only button that can save BeepBox songs. The songs are stored in a joint location that can be accessed through every project. If you try to save a song with a name that already exists, the old song will be replaced without warning.

Load Button

When pressed, the current song will be replaced with the desired song.

Build Button

The only button without a popup, this button imports the current BeepBox song into the Scratch sound layers.

Popup

Clicking the extension icon will bring up a popup of all availiable songs created and saved through Felis Beep. You can open these songs in BeepBox or delete them.
